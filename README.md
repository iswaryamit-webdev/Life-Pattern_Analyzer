# Life-Pattern_Analyzer
AI Life Pattern Analyzer

Overview

AI Life Pattern Analyzer is a full-stack web application that helps users track daily routines and analyze productivity patterns. The system provides insights through dashboards, charts, PDF reports, and Excel exports.

Features

- User Registration and Login
- Daily Routine Logging
- Dashboard Analytics
- Productivity Score Calculation
- Sleep Trend Visualization
- Mood Trend Visualization
- Search Logs by Date
- Edit and Delete Logs
- PDF Report Generation
- Excel Export Functionality
- User Profile Management

Technology Stack

- Python
- Flask
- MySQL
- HTML
- CSS
- Chart.js
- OpenPyXL
- ReportLab
